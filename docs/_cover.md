# retold-remote-ios

> A native iOS client for Retold Remote.

A Swift/SwiftUI application that connects to a `retold-remote` server to provide secure, on-the-go access to Retold ecosystem services -- dashboards, data records, file transfers, and live notifications.

- **Native SwiftUI** -- iOS 16+, iPadOS, and Mac Catalyst
- **Retold Remote client** -- talks to the `retold-remote` Orator API
- **Tidings realtime** -- live updates over WebSockets
- **Secure by default** -- Keychain-backed credential storage, biometric unlock
